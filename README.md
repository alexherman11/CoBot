# CoBot

This project is being developed for a 4th grade class to help them unlock
the power of Large Language Models in the devlopment of programs for 
the second generation VEX IQ Robots.

### The goals of this project are as follows:
-   To allow students to interact with the VEX IQ Robots using LLMs in 
    a controlled environment where teachers can manage and control 
    the amount of automation and feedback
-   To give teachers a tool to easily assess the performance and progress as 
    students are allowed to code at their own rate, and learn at their own pace
-   To demonstrate to faculty and students the incredible power of AI to realize and
    implement their most creative ideas without extensive training and prior knowledge.

### Interface Layout

The interface is friendly and designed to minimize distractions. The primary functionality included is as follows

#### AIBuild
The students can write natural language prompts into a box on the left side of the screen, and the python (auto-generated by the LLM) shows up in real time on the right side. Students have the ability to add features, modify the code, and ask questions about it, and make edits as they wish. This functionality is unlocked through the completion of eductional levels that they complete througout the program
